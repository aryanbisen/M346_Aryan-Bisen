## Erklärungen zum DNS
DNS staht für Domain Name System. Es ermöglicht die Umwandlung von Domain-Namen in IP-Adressen. Ohne DNS müssten Benutzer IP-Adressen direkt eingeben, um auf Websites zuzugreifen.
